# MopUE5 とは
MOPProtocol を受信してモーションを反映するライブラリです。
A library that receives MOPProtocol and shows motion in Unreal.

## 動作環境 / Supported OS/Unreal
- macOS
- UE5.0.2

## 開発環境 / Developed On

- UE5.0.2
